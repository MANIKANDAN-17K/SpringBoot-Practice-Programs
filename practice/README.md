# Spring Boot Practice Programs

A collection of 30 Spring Boot practice programs designed to learn backend development, REST APIs, database integration, security, and Spring Boot-specific features through hands-on examples.

## Topics Covered

### Backend Fundamentals (20)

1. Hello World REST API
2. GET endpoint returning static JSON
3. GET endpoint by ID
4. POST endpoint with request body
5. PUT/PATCH endpoint
6. DELETE endpoint
7. Proper HTTP status codes
8. Request validation
9. Global exception handling
10. Database integration
11. Pagination and sorting
12. Filtering with query parameters
13. One-to-Many relationship
14. Password hashing
15. JWT authentication
16. JWT authorization middleware/filter
17. Role-based access control
18. Request logging
19. CORS configuration
20. Environment-based configuration

### Spring Boot Specific (10)

21. JPA Entity and Repository
22. Many-to-Many Relationship
23. Transaction Management
24. Controller Advice & Exception Handling
25. Bean Validation with @Valid
26. DTO Layer Mapping
27. Spring Security Configuration
28. Spring Profiles
29. Scheduled Tasks
30. OpenFeign Client

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/springboot-practice-programs.git
```

Navigate into the project

```bash
cd springboot-practice-programs
```

Build the project

```bash
mvn clean install
```

Run the application

```bash
mvn spring-boot:run
```
